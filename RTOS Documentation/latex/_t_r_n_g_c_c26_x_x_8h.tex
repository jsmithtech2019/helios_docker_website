The T\+R\+N\+G\+C\+C26\+XX driver provides reentrant access to the T\+R\+NG module within the C\+C26\+XX. Reentrant access is controlled by creating a global critical section when requesting a random number. This critical section disables all Hwi\textquotesingle{}s from running.\hypertarget{_t_r_n_g_c_c26_x_x_8h_autotoc_md91}{}\doxysubsubsection{General Behavior}\label{_t_r_n_g_c_c26_x_x_8h_autotoc_md91}

\hypertarget{_t_r_n_g_c_c26_x_x_8h_autotoc_md94}{}\doxyparagraph{Initializing client parameters}\label{_t_r_n_g_c_c26_x_x_8h_autotoc_md94}

\begin{DoxyItemize}
\item Client parameters are optional. The Driver may be called with a N\+U\+LL pointer to the client parameters, in which case it will use the default parameters. The default parameters configurate the T\+R\+NG to produce 256 samples per cycle at 1 clock per sample. a call to T\+R\+N\+G\+C\+C26\+X\+X\+\_\+get\+Number with default values has a latency of 14us. Increasing any of the configuration parameters will result in an increase in latency.
\item The application initializes its parameters by calling \mbox{\hyperlink{group___t_r_n_g___s_t_a_t_u_s_ga71e12002e2dec081459ef9fd328e1890}{T\+R\+N\+G\+C\+C26\+X\+X\+\_\+\+Params\+\_\+init()}} and passing a pointer to a \mbox{\hyperlink{struct_t_r_n_g_c_c26_x_x___params}{T\+R\+N\+G\+C\+C26\+X\+X\+\_\+\+Params}} variable as an argument to it.
\item This function can be called once or more to each initialization of parameters.
\item This function can be called once or more with the same \mbox{\hyperlink{struct_t_r_n_g_c_c26_x_x___params}{T\+R\+N\+G\+C\+C26\+X\+X\+\_\+\+Params}} variable.
\item The client parameter are initialized to valid default values, but may be changed by the client after this function returns. See trng.\+h for full details.
\end{DoxyItemize}
