\doxysubsection{Detailed Description}
Display Global configuration. 

The \mbox{\hyperlink{struct_display___config}{Display\+\_\+\+Config}} structure contains a set of pointers used to characterize the Display interface implementation.

This structure needs to be defined before calling \mbox{\hyperlink{_display_8h_a3aa87973a354d4fd6a2969f764e8afe6}{Display\+\_\+open()}} and it must not be changed thereafter.
