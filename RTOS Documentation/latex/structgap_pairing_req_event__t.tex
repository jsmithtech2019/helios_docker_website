G\+A\+P\+\_\+\+P\+A\+I\+R\+I\+N\+G\+\_\+\+R\+E\+Q\+\_\+\+E\+V\+E\+NT message format.~\newline
 ~\newline
 This message is sent to the app when an unexpected Pairing Request is received. The application is expected to setup for a Security Manager pairing/bonding.~\newline
