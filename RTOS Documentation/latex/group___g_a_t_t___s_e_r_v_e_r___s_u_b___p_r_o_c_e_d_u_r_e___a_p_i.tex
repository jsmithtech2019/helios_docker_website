This sub-\/procedure is used when a server is configured to indicate a characteristic value to a client and expects an attribute protocol layer acknowledgement that the indication was successfully received. 

The A\+TT Handle Value Indication is used in this sub-\/procedure.

If the return status from this function is S\+U\+C\+C\+E\+SS, the calling application task will receive an O\+S\+AL G\+A\+T\+T\+\_\+\+M\+S\+G\+\_\+\+E\+V\+E\+NT message. The type of the message will be A\+T\+T\+\_\+\+H\+A\+N\+D\+L\+E\+\_\+\+V\+A\+L\+U\+E\+\_\+\+C\+FM.

Note\+: This sub-\/procedure is complete when A\+T\+T\+\_\+\+H\+A\+N\+D\+L\+E\+\_\+\+V\+A\+L\+U\+E\+\_\+\+C\+FM (with S\+U\+C\+C\+E\+SS or ble\+Timeout status) is received by the calling application task.
