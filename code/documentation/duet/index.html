Insert Customer data into MySQL Database Function will parse the data from the received JSON payload and then upload it into the MySQL database.

◆ upload_data()

def duet.upload_data ( )

Upload Endpoint of DUET The upload endpoint is where all requests from the controller applications will be sent.

This endpoint expects a JSON payload with valid credentials for the dealership, module and employee before inserting the data. Received data is also parsed and sanitized for security purposes.
