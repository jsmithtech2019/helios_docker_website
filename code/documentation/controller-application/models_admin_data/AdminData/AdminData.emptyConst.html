AdminData.emptyConst constructor

AdminData.emptyConst()

Default empty constructor

Generates an AdminData object with no values for any of the internal variables. This allows these values to be added to the object during runtime rather than at initialization of the object.

Implementation

AdminData.emptyConst() {
  this._employeeName = '';
  this._employeePhoneNumber = '';
  this._employeeEmail = '';
  this._employeePass = '';
  this._employeeUUID = '';
  this._moduleUUID = '';
  this._dealership = '';
  this._dealershipUUID = '';
}
