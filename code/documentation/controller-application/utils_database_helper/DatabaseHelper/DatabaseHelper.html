DatabaseHelper constructor

DatabaseHelper()

Protect against recreating the DatabaseHelper singleton.

Factory contructor for the DatabaseHelper object that verifies if a database exists within the file system already and protects from regenerating and overwriting existing saved data.

Function returns a DatabaseHelper singleton which is stored in the GetIt object and can be utilized to access the database and perform necessary functions to the data within it including:

  • Insertions
  • Selects
  • Updates
  • Deletes

Implementation

factory DatabaseHelper() {
  if (_databaseHelper == null) {
    _databaseHelper = DatabaseHelper._createInstance(); // This is executed only once, singleton object
  }
  return _databaseHelper;
}
